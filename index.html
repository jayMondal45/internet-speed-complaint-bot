Run Bot Locally

1

Clone the Repository

Get started by cloning the repository to your local machine:

git clone https://github.com/jaymondal45/internet-speed-complaint-bot.git
2

Navigate to Directory

Change into the project directory:

cd internet-speed-complaint-bot
3

Open in Browser

Simply open the index.html file in your preferred browser, or use a local server:

# Using Python
python -m http.server 8000

# Or just open index.html directly in your browser
